# Legal Nurse Consulting Platform - Backend

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Modules
1. CRM & Case Intake
2. Medical Records Organization
3. OCR & Intelligent Search
4. Timeline & Chronology Builder
5. Case Analysis & Standards of Care
6. Damages & Injury Tracking
7. Court-Ready Reports & Summaries
8. Secure File Sharing & Client Portal
9. Task, Workflow & Deadline Management
10. Time Tracking & Billing Engine
11. Collaboration & Internal Notes
12. Compliance, Security & Audit Controls
13. Analytics & Business Insights
14. Integrations & System Administration

## Setup
```bash
npm install
cp .env.example .env
# Configure .env file
npm run dev
```
