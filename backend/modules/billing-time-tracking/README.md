# Billing & Time Tracking Module

## Overview
Complete time tracking and billing system for legal nurse consulting services.

## Features
- Manual time entry and timer functionality
- Invoice generation from time entries
- Payment tracking and status management
- Revenue and aging reports
- User productivity analytics

## API Endpoints

### Time Entries: `/api/time-entries`
### Time Tracking: `/api/time-tracking`
### Invoices: `/api/invoices`
### Billing Analytics: `/api/billing`
